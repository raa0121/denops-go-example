denops-go-example
=========
denops plugin build by go(wasm).

Usage:
------
```
:GoAdd 1 1
```

Requirements:
-------------
[denops.vim](https://github.com/vim-denops/denops.vim)

Install:
--------
```
Plug 'raa0121/denosp-go-example'
```

How To build wasm:
------------------
Windows
```
build.bat
```

macOS/Linux
```
./build.sh
```
